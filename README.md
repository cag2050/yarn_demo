# yarn_demo
Facebook推出的Yarn：为速度而打造的开源JavaScript包管理器，或取代npm    

依次运行下列命令：  
npm install -g yarn    
yarn init       
yarn add gulp    
yarn add browser-sync --dev    

与npm的等价命令：    
npm install === yarn   
npm init === yarn init  
npm install taco --save === yarn add taco   
npm uninstall taco --save === yarn remove taco  
npm install taco --save-dev === yarn add taco --dev
